# Auto-tests for various smart-contracts
![build status](https://github.com/ZLATbI4/smart_contracts_autotests/actions/workflows/docker-build.yml/badge.svg)

### How to run tests?

**By shell:**

```shell
bash run_tests.sh
```

**Or via docker:**

```shell
docker build -t soltests .
docker run soltests
```
